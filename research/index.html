Accessibility Review

There are no major accessibility issues on this webpage. There were zero errors found, 2 contrast errors, and 2 alerts.
The most significant issues were 2 low contrast errors and the alert regarding long alternative text. The images do have
alternative text descriptions. One specific actionable change would be to improve the contrast between the Center for
Social Work Research heading and the image that serves as the background to it by lowering the images brightness.

HTML Validation

There was a single error shown regarding an element being hidden but also being on the noscript element.
This creates an issue for users who have JavaScript disabled because they will not be able to access the
content within the noscript element. This should be corrected within the code by changing the element's attributes.
